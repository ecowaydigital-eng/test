# Домёк — лендинг

Лендинг центра раннего развития «Домёк». Статичный сайт: HTML + CSS + немного ванильного JS, без сборки и зависимостей.

## Структура

```
domek-landing/
├── index.html          # разметка страницы
├── css/
│   └── styles.css      # все стили
├── js/
│   └── script.js       # бургер-меню + модальная форма записи
├── images/             # все изображения (фото, иконки, персонаж, маршрут)
└── README.md
```

## Блоки

1. Hero — «Когда развитие ребёнка вызывает вопросы»
2. «Возможно, вы узнаете свою ситуацию»
3. «Первые годы жизни имеют большое значение»
4. «Как проходит наша работа с семьёй» (маршрут из 5 этапов)
5. «Одна команда вокруг ребёнка» (команда + 6 специалистов)
6. CTA «Готовы обсудить развитие ребёнка?» + модальная форма записи

## Запуск

Локально — просто откройте `index.html` в браузере. Либо поднимите статический сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## Публикация на GitHub Pages

1. Запушьте папку в репозиторий.
2. Settings → Pages → Source: ветка `main`, папка `/root`.
3. Сайт будет доступен по адресу `https://<username>.github.io/<repo>/`.

## Зависимости

Только шрифты Google Fonts (Manrope, Inter, Nunito) подключаются по CDN в `<head>`.
Всё остальное — локальные файлы, интернет для работы сайта не требуется (кроме шрифтов).
